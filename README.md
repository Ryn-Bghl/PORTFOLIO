# Rayan BOUGHALIA / Portfolio

This is the repository for my personal portfolio website. It showcases my skills, projects, and contact information.

## 🚀 Live Demo

You can view the live version of my portfolio [here](https://ryn-bghl.github.io/PORTFOLIO/).

## 📖 About

This portfolio is a static website built with HTML, CSS, and JavaScript. It is designed to be a clean, modern, and responsive showcase of my work as a web developer. The website is available in both English and French.

## ✨ Features

*   **Responsive Design:** The website is fully responsive and works on all devices.
*   **Multi-language Support:** The website is available in English and French.
*   **Project Showcase:** A dedicated section to showcase my projects with links to live demos and source code.
*   **Contact Form:** A contact form that opens the user's default mail client.
*   **Skills Section:** A section to display my skills and technologies I'm proficient in.

## 🛠️ Technologies Used

*   **CSS3**
*   **JavaScript (ES6)**
*   **Git**
*   **GitHub Pages**
